Background_Blend_mode

با استفاده از ویژگی background-blend-mode در CSS میتوان رنگ زمینه را با تصویر زمینه مخلوط یا ترکیب کرد، یعنی مثلا اگه رنگ زمینه قرمز باشه میتونیم همین رنگ رو با تصویر زمینه مخلوط کنیم. در نتیجه میشه تصویر زمینه را بصورت رنگی نمایش داد، بطور کلی با استفاده از این ویژگی میتوان افکت های رنگی را بروی تصاویر قرار داد.
